# CARE-IT Kernartefakte

Die Kernartefakte bilden das operative Fundament des Frameworks.

Dazu zählen:

- Systemverbund-Beschreibung  
- Rollen- und Verantwortungsmatrix  
- Lifecycle-Übersicht  
- Impact-Check  
- Innovations-Canvas  

Sie sind aufeinander abgestimmt
und unterstützen die praktische Anwendung
der CARE-IT Prinzipien und Domänen.

Organisationen können einzelne Artefakte
schrittweise einführen,
ohne das gesamte Modell auf einmal umzusetzen.
